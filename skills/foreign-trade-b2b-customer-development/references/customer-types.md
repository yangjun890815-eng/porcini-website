# 客户类型判断

判断客户类型时，先看官网证据，再做业务判断。

## 常见客户类型

| 类型 | 典型官网线索 |
|---|---|
| 进口商 | import, sourcing, global suppliers, overseas manufacturers |
| 经销商 | distributor, dealer network, brands represented, authorized distributor |
| 批发商 | wholesale, trade customers, bulk orders, B2B pricing |
| 零售商 | stores, shop online, retail locations, consumer products |
| 品牌商 | own brand, product lines, brand story, private label |
| OEM/ODM 买家 | private label, custom products, OEM, ODM, contract manufacturing |
| 工程商/项目商 | projects, installation, contractor, case studies, tenders |
| 代理商 | agent, representative, sales agent, territory |
| 电商卖家 | online store, marketplace store, DTC brand |
| 系统集成商 | integration, solutions, engineering, system design |
| 贸易商 | trading, export/import, broad product categories |
| 暂不确定 | 信息不足，官网没有清晰业务模式 |

## 判断规则

- 一个客户可以有多个类型，例如“品牌商 + 零售商”。
- 不要只因为公司名带 Trading 就判断为贸易商，要看官网业务。
- 不要只因为有 Shop 就判断为零售商，要看是否真实销售目标产品。
- 如果官网打不开或信息很少，写 `暂不确定`。
