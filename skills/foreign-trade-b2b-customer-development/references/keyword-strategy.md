# 关键词策略

## 组合方式

不要只搜一个词。推荐组合：

```text
产品词 + 客户类型词 + 国家/地区
产品词 + distributor/importer/wholesaler + country
产品词 + dealer/brand/retailer + country
产品词 + local language keyword + country
```

```text
importer
```


## 本地语言

根据目标市场加入当地语言词。比如德国用 German，法国用 French，西班牙用 Spanish。不要只用英文。
